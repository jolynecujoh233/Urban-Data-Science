# Urban-Data-Science
Advanced Urban Data Science helps you deepen and expand the foundations of urban data science.

## 01 Exploratory Data Analysis (EDA)
This section mainly covers:   
**Importing and inspecting data:**   
Load the dataset and check its structure and variable types.   
**Data cleaning and preparation:**   
Handle missing values, select relevant variables, and convert data types.   
**Exploratory Data Analysis (EDA):**   
Use summary statistics and simple visualizations to understand data distribution.   
**Basic analysis or modeling:**   
Apply statistical methods (e.g., regression or grouped summaries) to examine relationships.   
The overall workflow is:
Import → Clean → Explore → Analyze
